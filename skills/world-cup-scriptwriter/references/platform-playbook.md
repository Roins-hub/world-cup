# Platform Playbook

## Xiaohongshu

Best for: football beginners, lifestyle participation, watch-party identity, useful carousels.

Use angles:

- 不懂球也能看懂
- 今晚看这 3 个点
- 主队配色 OOTD
- 看球搭子别接不上这些梗
- 这不是足球，这是成年人情绪片

Format:

- Carousel: cover claim -> one-line conclusion -> timeline -> people/terms -> “how to talk about it”.
- Short video: hook -> context -> one emotional or useful takeaway -> comment question.

## Douyin

Best for: conflict, fast explanation, controversy, “what happened”.

Use angles:

- 最后20分钟发生了什么？
- 这球到底该不该判？
- AI重新模拟世界杯，冠军变了
- 大热必死？先排除这几支热门

Format:

- 0-3s: claim/question.
- 3-15s: event replay in plain language.
- 15-35s: reason/controversy.
- final: choose a side.

## Weibo

Use as radar, not final script format. Convert hot words into script triggers:

- 红牌/点球/VAR -> controversy explainer.
- 绝杀/逆转/最后20分钟 -> emotion clip.
- 球星名字 -> personality story.

## Bilibili

Use for longer emotional reference. Borrow rhythm, not material:

- 4K/high-energy montage -> pacing and music arc.
- “足球的力量” style -> human stakes and memory.
