# Script Patterns

## Beginner Explainer

Hook: `不懂球也没关系，今晚你只看一个问题就够了。`

Flow:

1. Translate the match into one human question.
2. Explain the pressure in plain language.
3. Show what visual proof to use.
4. Give the viewer a phrase they can repeat.

## Controversy Explainer

Hook: `这球到底冤不冤，先别急着站队。`

Flow:

1. Name the disputed moment.
2. Give both sides.
3. Translate the rule into a simple condition.
4. Ask the audience to choose.

## Emotional Moment

Hook: `先看旧镜头。这场球的情绪，是从过去带过来的。`

Flow:

1. Open with the emotional claim.
2. Give one past wound or pressure point.
3. Bind it to a face/crowd/action shot.
4. End with miracle vs collapse.

Use one real old fact instead of a broad phrase like “命运感”.

## Replay-Led Prediction

Hook: `这场别先猜比分，先找一个会被反复回放的细节。`

Flow:

1. Name the visible clue before the prediction.
2. Explain why the first glance is incomplete.
3. Give the viewer one lens: replay, first rushed touch, first counterattack, face, crowd, or body language.
4. Name one player in Chinese if possible.
5. End with a comment-side choice.

Example:

`西班牙对沙特，真正能出圈的可能不是比分，是旧账被翻出来那一刻。`

`沙特赢过阿根廷，西班牙也吃过被反打的亏。两段旧镜头一接，今天这场就不是普通赛前分析了。`

`你盯亚马尔第一次被包夹后的反应。是继续要球，还是回头找人，这个镜头比控球率更好讲。`

`萨勒姆·达瓦萨里也不需要整场抢戏。一次断球，一脚往前给，只要让看台声音变一下，故事就立住了。`

`这种球，小红书观众先记住的往往不是阵型，是表情、回放，还有谁在那一秒被看错了。`

`你觉得这场最该盯谁的第一个小动作？我会先看亚马尔。`

## Underdog Hero

Hook: `所有人都觉得他只是来陪跑的。`

Flow:

1. Open with “nobody believed this person/team”.
2. Give one hard contrast: age, population, value gap, old upset, saves, or possession.
3. Name the pressure on the stronger side.
4. Tell non-fans what to watch.
5. End with “can they do it again?”.

Example for Spain vs Saudi Arabia:

`所有人都觉得沙特只是来陪跑的。可世界杯最吓人的地方，就是它真给小球队留门缝。`

`四年前，沙特就是从这条门缝里钻出来，2 比 1 掀翻了阿根廷。那天之后，没人敢再把他们当背景板。`

`现在换成西班牙站在他们面前。西班牙最会控球，也最怕一种局面：球一直在脚下，比分却一直打不开。`

`沙特不需要全场压着打。他们只等一次断球，一次边路冲刺，一次让看台突然安静的反击。`

`所以这场开局别只看谁拿球多。你就看第一次反击之后，西班牙球员有没有回头找人。`

`如果慢镜头里能看到那一下犹豫，这条视频就有了评论区。你觉得沙特还能再吓强队一次吗？`

## Lifestyle Participation

Hook: `比赛还没开踢，但今晚的气氛已经可以先发了。`

Flow:

1. Watch-party reason.
2. Outfit/night snack/friend angle.
3. One team or player story.
4. Ask how the viewer joins the atmosphere.

## Rewrite Examples

Machine: `本场比赛双方历史表现都值得关注。`

Human: `别急着猜比分，先看谁更怕输。`

Machine: `世界杯最会制造轮回感，今天这场刚好有老素材可以托底。`

Human: `沙特赢过阿根廷，西班牙被日本逆转过。把这两段旧镜头接上，今天这场就有了开头。`

Machine: `该素材可以用于剪辑赛前氛围。`

Human voiceover: `比赛还没开踢，但气氛已经先起来了。你看球衣、看看台、看主帅说话前停的那一下，就知道今晚有多紧。`

Editor note: `球迷、发布会表情、队徽和入场镜头放在 visualInstruction，不要进入 voiceover。`

Machine: `这场比赛将为观众提供参考。`

Human: `你今晚不懂球，也能靠这三个镜头跟上评论区。`
